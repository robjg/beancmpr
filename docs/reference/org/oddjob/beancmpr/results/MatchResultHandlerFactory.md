# beancmpr:match-results

Creates simple result objects. The results are created from the
comparison events generated by the [beancmpr:compare](../../../../org/oddjob/beancmpr/BeanCompareJob.md)
and forwarded to a downstream consumer that will typically report on them in some
way.

### Property Summary

| Property | Description |
| -------- | ----------- |
| [ignoreMatches](#propertyignorematches) | If true, then matches will be filtered out so only differences will be reported. | 
| [results](#propertyresults) | If no consumer is provided result will be captured in this list. |
